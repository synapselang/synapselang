# Roadmap

## Phase 1
- Lexer
- Parser for a useful subset
- AST
- CLI for tokens and AST printing

## Phase 2
- Runtime evaluator for scalars, let bindings, blocks, and built-ins
- NumPy-backed TensorValue
- Core tensor operations

## Phase 3
- Reverse-mode autodiff
- `grad` and `value_and_grad`
- Structured gradients for structs and tuples

## Phase 4
- Resolver and stronger type checking
- Shape inference and better diagnostics
- Imports and module loading

## Phase 5
- REPL
- Expanded standard library
- More examples and test coverage
